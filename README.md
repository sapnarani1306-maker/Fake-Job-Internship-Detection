# 🕵️ Fake Internship Detection System

## 📌 Project Overview
This project is a Machine Learning web application that detects whether a job/internship posting is **Genuine or Fake**.  
It uses **TF-IDF Vectorization** and **Logistic Regression** to analyze job descriptions.

---

## 🚀 Features
- Detects fake vs genuine internship postings
- Simple Streamlit web interface
- Real-time prediction
- Model trained using ML pipeline
- Easy to use UI

---

## 🧠 Machine Learning Model
- Algorithm: Logistic Regression  
- Vectorizer: TF-IDF  
- Accuracy: ~95%
